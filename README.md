react移动端脚手架：
  react + webpack + dva + antd-mobile

线上使用：m.v.huatu.com



### Run the example

```
npm install

npm start

npm run dev:test 本地测试环境

npm run dev:pro 本地正式环境

npm run pro 打包正式环境

npm run test 打包测试环境

```

Browser should open automatically. Otherwise, navigate to the URL reported in the terminal
